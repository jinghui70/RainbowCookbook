简介

Rainbow 开发平台是一个基于 java 的业务开发平台，它的设计原则是尽可能的简单，其设计目标如下：

  • 解耦代码，易于扩展
  • 数据库无关
  • 面向服务开发
  • 开发简单，聚焦于业务
  • 测试简单，保证代码质量
  • 部署简单，支持局部代码热更新
  • 让开发人员聚焦于业务开发，不用在系统的管理配置上浪费时间。
